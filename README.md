## 使用程序
### 程序如何运作？
程序会根据您的选择，自动按住Shift/Space/W键不放，直到您按下“停止运行”才会松开。  
### 如何安装程序？
该程序无需安装，您只需要下载EXE文件后双击运行即可  
### 软件兼容性如何？
理论上支持windows 10,11(64-bit)全版本  
本程序使用Python 3.10.9进行编译，Pyinstaller 5.0.4进行打包
